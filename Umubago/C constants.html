C constants

C constants:is an elemant whose value can not be changed at the time of execution of program is also known as constant and it may be int ,float and character data types.

Rules for constructing integer constant

  1. It must have at least one digit
  2. It must have a decimal point
  3. It may be positivr or negative
  4. No comma in floating point constant

Rules for constructing character constant

  1. Character constant is enclosed within quotes i.e (chat a='s';)
  2. It is a single alphabet,digit or special symbol
  3. The length of character is 1 character
